# مكونات تحليل Deepfake

تم تقسيم صفحة عرض التحليل إلى مكونات منفصلة لتحسين الصيانة والأداء.

## المكونات المتاحة

### 1. `general-info.blade.php`
**الوصف:** يعرض المعلومات العامة للتحليل
**المتغيرات المطلوبة:**
- `$analysis` - كائن التحليل

**المحتوى:**
- اسم الملف
- مستوى الثقة
- النتيجة (أصيل/مزيف)
- نوع الملف
- تاريخ التحليل
- المستخدم

### 2. `technical-details.blade.php`
**الوصف:** يعرض التفاصيل التقنية وإحصائيات المعالجة
**المتغيرات المطلوبة:**
- `$analysis` - كائن التحليل
- `$json` - البيانات المحللة من JSON
- `$mainResult` - النتيجة الرئيسية

**المحتوى:**
- معلومات النموذج المستخدم
- نوع التحليل
- الميزات المستخدمة
- إحصائيات المعالجة (وقت المعالجة، حجم الملف)
- إحصائيات خاصة بنوع الملف (وجوه، إطارات، مقاطع صوتية)

### 3. `detailed-analysis.blade.php`
**الوصف:** يعرض التحليل المفصل حسب نوع الملف
**المتغيرات المطلوبة:**
- `$analysis` - كائن التحليل
- `$rawJson` - البيانات الخام من JSON

**المحتوى:**
- جدول تفصيلي للصور (الوجوه المكتشفة)
- جدول تفصيلي للفيديو (الإطارات المحللة)
- جدول تفصيلي للصوت (المقاطع الصوتية)

### 4. `special-sections.blade.php`
**الوصف:** يعرض الأقسام الخاصة مثل سبب التصنيف والتحليل المفصل للصوت
**المتغيرات المطلوبة:**
- `$analysis` - كائن التحليل
- `$rawJson` - البيانات الخام من JSON

**المحتوى:**
- سبب التصنيف للصوت
- تحليل MFCC للمقطع الأكثر شكاً
- سبب التصنيف للمحتوى المزيف

### 5. `actions.blade.php`
**الوصف:** يعرض أزرار الإجراءات المتاحة
**المتغيرات المطلوبة:**
- `$analysis` - كائن التحليل

**المحتوى:**
- تحميل PDF
- طباعة التقرير
- تقييم النتيجة
- إبلاغ مزيف

### 6. `modals.blade.php`
**الوصف:** يعرض النوافذ المنبثقة والسكريبتات
**المتغيرات المطلوبة:**
- `$analysis` - كائن التحليل

**المحتوى:**
- نافذة عرض الصور مع إمكانية التكبير
- نافذة تقييم النتيجة
- سكريبتات JavaScript للتفاعل

### 7. `footer.blade.php`
**الوصف:** يعرض الفوتر مع معلومات النظام
**المتغيرات المطلوبة:**
- لا يحتاج متغيرات

**المحتوى:**
- معلومات النظام
- الميزات الرئيسية

## كيفية الاستخدام

في الملف الرئيسي `show.blade.php`:

```blade
<!-- General Information Section -->
<x-analysis.general-info :analysis="$analysis" />

<!-- Technical Details Section -->
<x-analysis.technical-details :analysis="$analysis" :json="$json" :mainResult="$mainResult" />

<!-- Detailed Analysis Section -->
<x-analysis.detailed-analysis :analysis="$analysis" :rawJson="$rawJson" />

<!-- Special Sections -->
<x-analysis.special-sections :analysis="$analysis" :rawJson="$rawJson" />

<!-- Action Buttons -->
<x-analysis.actions :analysis="$analysis" />

<!-- Footer -->
<x-analysis.footer />

<!-- Modals and Scripts -->
<x-analysis.modals :analysis="$analysis" />
```

## المزايا

1. **سهولة الصيانة:** كل مكون منفصل ويمكن تعديله بسهولة
2. **إعادة الاستخدام:** يمكن استخدام المكونات في صفحات أخرى
3. **أداء أفضل:** تحميل أسرع للصفحة
4. **تنظيم أفضل:** كود أكثر تنظيماً ووضوحاً
5. **اختبار أسهل:** يمكن اختبار كل مكون على حدة

## ملاحظات مهمة

- تأكد من تمرير جميع المتغيرات المطلوبة لكل مكون
- جميع المكونات تدعم التأثيرات البصرية (AOS)
- المكونات متجاوبة مع جميع أحجام الشاشات
- تم الحفاظ على جميع الوظائف الأصلية
